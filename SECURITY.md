# Security Policy

## Supported Versions

This section lists the versions of this project that are currently receiving security updates.

| Version | Supported          |
| ------- | -----------------  |
| 0.1.x   | :white_check_mark: |
| < 0.0.1 | :x:                |

> Note: Only the latest stable release in each major version is actively supported.  
> Older or deprecated versions may not receive security fixes.

## Security Best Practices for Users

- Keep your project dependencies up-to-date.
- Regularly update to the latest supported versions.
- Review release notes for security patches and advisories.
